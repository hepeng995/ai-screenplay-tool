# Demo 视频脚本

## 视频信息

- **时长**：5-10 分钟
- **分辨率**：1920×1080 (1080p)
- **格式**：MP4
- **录制工具**：OBS Studio
- **音频**：外接麦克风录音

## 旁白稿

### 开场（30s）

> 大家好！今天给大家演示的是「AI 剧本工坊」——一款将小说文本自动转换为 YAML 格式剧本的 AI 辅助创作工具。
>
> 这个项目是七牛云校园编程大赛的参赛作品，使用 Next.js 14 全栈框架开发，集成 mimo-v2.5 AI 模型和七牛云对象存储，零成本部署在 Vercel 平台。

### 功能演示 - 上传（45s）

> 首先看到的是首页。点击「开始创作」进入上传页面。
>
> 支持拖拽上传和点击选择两种方式，接受 .txt 和 .md 格式，单文件限制 10MB。
>
> 我这里上传一份测试小说文件，可以看到文件名、大小和字符数立即显示出来了。

### 功能演示 - 章节切分（45s）

> 上传完成后，系统自动识别小说的章节边界。
>
> 使用正则表达式匹配中英文章节标记，包括"第一章"、"Chapter 1"、"第一回"等多种格式。要求至少识别 3 个章节才能继续。
>
> 可以看到这个测试文件被正确切分成了多个章节，每个章节显示了标题和字数。

### 功能演示 - AI 转换（1min30s）

> 点击「开始 AI 转换」，系统会将每个章节独立发送给 mimo-v2.5 大模型。
>
> 关键技术点：每次请求使用 Edge Runtime，超时限制 25 秒。AI 调用包含重试机制，最多重试 2 次，采用指数退避策略。
>
> 可以看到进度条实时更新。转换完成后会自动跳转到编辑器页面。

### 功能演示 - YAML 预览（45s）

> 编辑器页面采用左右双栏布局。右侧是 YAML 预览区。
>
> 自动解析 YAML 内容，以树形结构展示幕、场景、台词的层次关系。顶部显示统计数据：幕数、场景数、台词数、角色数。
>
> 每个节点都可以展开折叠，方便浏览。

### 功能演示 - 在线编辑（45s）

> 左侧是编辑区。使用 TextArea 编辑器，支持实时校验。
>
> 每次输入后 500 毫秒自动触发 Zod Schema 校验和 YAML 语法校验。校验通过显示绿色标记，失败时显示具体错误位置。
>
> 编辑器支持 Ctrl+S 手动保存，每 5 秒自动保存到 localStorage。

### 功能演示 - 导出（30s）

> 点击「导出」下拉菜单，支持 YAML 和 JSON 两种格式。
>
> YAML 格式直接下载当前编辑器内容。JSON 格式会先解析 YAML 再序列化为格式化的 JSON。
>
> 文件名格式为：script-项目ID-日期.yaml

### 技术亮点（1min）

> 项目的技术亮点：
>
> 1. **Edge Runtime**：AI 转换使用 Vercel Edge Runtime，全球 CDN 加速，响应更快。
> 2. **Zod 双重校验**：前后端共享 Schema 定义，实时 YAML 结构校验。
> 3. **并发控制**：限制最多 3 个并发 AI 请求，防止 API 过载。
> 4. **零成本架构**：Vercel Hobby + 七牛云免费额度 + localStorage，无服务器成本。
> 5. **完整测试体系**：17+ 单元测试覆盖核心工具函数。

### 总结（30s）

> 项目地址在 GitHub：github.com/hepeng995/ai-screenplay-tool
>
> 技术栈：Next.js 14 + Tailwind CSS + Zod + mimo-v2.5 + 七牛云
>
> 感谢观看！

## 录制建议

1. **环境准备**：关闭通知，准备测试数据
2. **录音**：使用外接麦克风，避免环境噪音
3. **后期**：添加字幕（可选），调整音量
4. **上传**：B 站或腾讯视频
