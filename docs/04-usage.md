# 用户使用指南

## 快速开始（3 步完成）

### 第 1 步：上传小说

1. 打开首页，点击「开始创作」
2. 在上传页面，拖拽 `.txt` 或 `.md` 文件到虚线区域
   - 支持 UTF-8 编码
   - 文件大小 ≤ 10MB
   - 需包含 3 个以上章节（如「第一章」「Chapter 1」等）
3. 确认文件信息后点击「下一步」

### 第 2 步：章节切分 + AI 转换

1. 系统自动识别章节标记并展示切分结果
2. 确认章节数量和内容后，点击「开始 AI 转换」
3. 等待 AI 处理（每章约 20-90 秒）
4. 转换完成后自动跳转到编辑器

### 第 3 步：编辑 + 导出

1. 在编辑器左侧修改 YAML 内容
2. 右侧实时预览树形结构（幕 → 场 → 台词）
3. 校验错误会在编辑器中红色提示
4. 点击「导出」选择 YAML 或 JSON 格式下载

## 进阶功能

### 云端同步（七牛云）

| 操作 | 按钮 | 说明 |
|------|------|------|
| 上传云端 | ☁️ 上传云端 | 将当前 YAML 保存到七牛云 |
| 云端载入 | ☁️ 云端载入 | 从七牛云下载之前保存的文件 |

### 自动保存

- 编辑器每 5 秒自动保存到浏览器 localStorage
- 按 `Ctrl+S` 手动保存

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+S | 手动保存 |

## YAML 剧本结构

```yaml
script:
  title: "小说标题"
metadata:
  genre: "类型"
  characters: ["角色A", "角色B"]
  summary: "一句话简介"
acts:
  - act_number: 1
    title: "第一幕"
    scenes:
      - scene_number: 1
        location: "场景地点"
        dialogues:
          - character: "角色A"
            type: "对白"
            content: "台词内容"
```

详细 Schema 定义请参考 [05-yaml-schema.md](./05-yaml-schema.md)

## 常见问题

### Q: AI 转换太慢？

每章约 20-90 秒，大文件请耐心等待。系统已使用 Edge Runtime 加速。

### Q: 上传后内容丢失？

检查浏览器 localStorage 是否被清除。建议重要内容上传到七牛云。

### Q: 支持哪些文件格式？

仅支持 `.txt` 和 `.md` 纯文本文件（UTF-8 编码）。

### Q: 章节切分不准？

系统支持「第X章」「Chapter N」等标准格式。非标准格式请手动调整。
